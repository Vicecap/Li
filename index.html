Match Widget

Embed live sports match data anywhere with simple HTML

How to Use

1. Add the widget container with team data attributes:

<div class="match-widget-container" data-team1="West Ham United" data-team2="Chelsea"></div>

2. Include the script from GitHub:

<script src="https://vicecap.github.io/Kingg/github-widget.js"></script>

3. That's it! The widget will automatically load and display match data.
